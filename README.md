**SDown Loader** is a powerful and user-friendly Spotify downloader that works on Windows, macOS, and Linux. Download your favorite playlists, albums, and single tracks with ease! 🚀

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 📋 **Playlist Download** | Download entire Spotify playlists with one click |
| 🎵 **Single Track** | Download individual songs |
| 💿 **Album Download** | Download full albums |
| 🔍 **Search & Download** | Search for songs and download directly |
| 📚 **Batch Download** | Download multiple URLs at once |
| 📂 **File Management** | View and manage downloaded files |
| 📜 **Download History** | Track all your downloads |
| 🎨 **GUI Version** | Beautiful graphical interface |
| 📦 **Multiple Formats** | Support for MP3, FLAC, M4A, WAV, OGG |
| ⚙️ **Advanced Settings** | Customize your downloads |
| 📋 **Clipboard Support** | Auto-detect Spotify links |

## 🚀 Quick Start

### 1️⃣ Installation

```bash
# Clone the repository
git clone https://github.com/zougar99/sdown-loader.git
cd sdown-loader

# Install dependencies
pip install -r requirements.txt
```

### 2️⃣ Install FFmpeg

| OS | Command |
|----|---------|
| 🍎 macOS | `brew install ffmpeg` |
| 🐧 Linux (Ubuntu) | `sudo apt install ffmpeg` |
| 🪟 Windows | `winget install ffmpeg` or [Download](https://www.gyan.dev/ffmpeg/builds/) |

### 3️⃣ Run the App

```bash
# 🎛️ Command Line Version
python SDown_Loader.py

# 🖥️ GUI Version (Graphical Interface)
python SDown_GUI.py
```

## ⚠️ Troubleshooting

### ❓ Common Issues

| Problem | Solution |
|---------|----------|
| FFmpeg not found | Install FFmpeg and add to PATH |
| Rate limit reached | Wait 24 hours or use Spotify Authentication |
| spotdl not installed | Run: `pip install --upgrade spotdl` |
| SSL certificate error | Update Python or use VPN |

### 🔧 Fix FFmpeg

```bash
# Windows (PowerShell)
winget install ffmpeg

# macOS
brew install ffmpeg

# Linux
sudo apt install ffmpeg
```
